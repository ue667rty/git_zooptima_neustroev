# Zoóptima — Ветеринарная клиника

## Описание проекта
Современный адаптивный веб-сайт для ветеринарной клиники «Zoóptima». Сайт включает все необходимые разделы для информирования клиентов и онлайн-записи на приём.

## Структура сайта
- **Главная страница** — приветствие, преимущества, превью врачей
- **Услуги и цены** — подробный прайс-лист по категориям
- **Наша команда** — информация о врачах (фото, специализация, стаж, график)
- **Контакты** — адрес, телефоны, схема проезда
- **Блог** — полезные статьи о здоровье животных
- **Новости** — анонсы акций и событий
- **Запись онлайн** — форма с календарём и выбором времени
- **Обратная связь** — форма для вопросов и предложений

## Технологии
- HTML5
- CSS3 (Flexbox, Grid, медиазапросы)
- JavaScript (календарь, валидация, маска телефона)
- PHP (обработка форм, отправка email, работа с БД)
- Font Awesome 6
